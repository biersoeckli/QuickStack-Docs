# QuickStack Documentation

QuickStack is a simple Web UI to manage your Linux server and deploy your containerized applications.
Think of QuickStack as your personal cloud platform, built to run on your servers.

## What is QuickStack?

At its core, QuickStack automates:

- **Building & Deploying Applications:** Deploy from Git repositories or Docker images to one server or a cluster.
- **Managing Applications:** Centralized web interface to monitor, backup, and manage your applications.

## Under the Hood

QuickStack uses these components:

| Component | Role |
|---|---|
| **[k3s](https://k3s.io/)** | Lightweight Kubernetes |
| **[Traefik](https://traefik.io/)** | Reverse proxy and load balancer |
| **[Longhorn](https://longhorn.io/)** | Distributed block storage |
| **[registry](https://hub.docker.com/_/registry)** | Docker registry |
| **[buildkit](https://github.com/moby/buildkit)** | Container image builder |

3. **[Expose your app with a domain + HTTPS](/docs/tutorials/expose-app)**.
